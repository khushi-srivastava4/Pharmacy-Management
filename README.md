#  Pharmacy Management System

A full-stack Pharmacy CRM built using FastAPI + React to manage inventory, sales, and purchase orders with real-time dashboard insights.



##  Features

###  Inventory Management
- Add medicines with:
  - Quantity
  - Expiry Date
  - MRP
  - Cost Price
- Auto status detection:
  - Active
  - Low Stock
  - Out of Stock
  - Expired

###  Sales Tracking
- Record medicine sales
- Real-time stock deduction
- Sales reflected in dashboard instantly

### Purchase Orders
- Create purchase orders
- Mark as completed
- Stock updates automatically

### Dashboard Insights
- Today's Sales
- Items Sold Today
- Purchase Today
- Low Stock Count
- Recent Sales Table



## Full Stack

### Frontend
- React
- Axios
- CSS

### Backend
- FastAPI
- SQLAlchemy
- SQLite
